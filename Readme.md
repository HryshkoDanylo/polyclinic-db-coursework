# Курсова робота  
## Розробка та аналіз надвеликої бази даних для предметної області «Поліклініка»

Цей репозиторій містить курсову роботу, присвячену проєктуванню, реалізації та аналізу надвеликої бази даних для предметної області **«Поліклініка»** з використанням технологій **Microsoft SQL Server**.

Робота охоплює повний цикл роботи з даними: від побудови операційної бази даних (OLTP) до створення сховища даних, реалізації ETL-процесів, побудови OLAP-куба та формування аналітичних звітів.

---

## Мета роботи

Метою курсової роботи є розробка надвеликої бази даних для медичної установи (поліклініки), а також реалізація інструментів аналітичної обробки даних з метою підтримки процесів аналізу та прийняття управлінських рішень.

---

## Предметна область

Предметна область охоплює основні бізнес-процеси поліклініки, зокрема:
- реєстрацію та облік пацієнтів;
- облік лікарів та їх спеціалізацій;
- фіксацію візитів пацієнтів;
- встановлення діагнозів;
- призначення лікарських засобів;
- аналіз статистичних та часових показників діяльності закладу.

---

## Використані технології

- **Microsoft SQL Server**
- **SQL Server Management Studio (SSMS)**
- **SQL Server Integration Services (SSIS)** — ETL-процеси
- **SQL Server Analysis Services (SSAS)** — OLAP-куб
- **SQL Server Reporting Services (SSRS)** — аналітичні звіти
